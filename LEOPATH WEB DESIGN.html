HTML5 MARKUP

It introduces several new elements and attributes that provide enhanced support for multimedia, graphics, and interactivity that help define the structure of a webpage more meaningfully. The adoption of HTML5 has significantly contributed to a richer and more interactive web experience, encouraging the development of feature-rich websites and web applications.

CSS STYLING

It enables web developers to control the appearance of elements on a webpage, such as fonts, colors, spacing, positioning, and more. By defining styles using CSS, developers can create consistent and visually appealing designs across different web pages and devices. CSS operates by associating style rules with HTML elements, allowing for separation between content (HTML) and presentation (CSS), which enhances the maintainability and flexibility of web design.

GRAPHIC DESIGN

It is the art of arranging elements harmoniously to create visually appealing and impactful designs across various mediums such as print, digital platforms, advertisements, branding materials, and more. Graphic designers use their artistic skills and design principles to convey concepts, evoke emotions, and enhance the overall aesthetic experience, while also considering the functional aspects of the design's intended purpose.
